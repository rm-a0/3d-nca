\section{Prerequisites}
This thesis assumes the reader is familiar with basic concepts 
of deep learning, in particular convolutional neural networks, gradient-based 
optimisation, and automatic differentiation in PyTorch \cite{PaszkePyTorch, PyTorchDocs}. 

A comprehensive introduction to these topics can be found in Goodfellow et al.~\cite{GoodfellowDeepLearning} 
and in the official PyTorch documentation \cite{PyTorchDocs}.

The rest of the thesis is self-contained with respect to cellular automata and their neural extensions.

\section{Cellular Automata}
Cellular automata (CA) are discrete computational systems that consist of a regular grid of cells, 
each of which can take on a finite number of states. The state of each cell evolves in parallel over discrete time steps
according to a local update rule, which depends on the states of its neighboring cells \cite{MarinescuComplexSystems}.
Formally, a cellular automaton is defined by:
\begin{itemize}
    \item A lattice of cells, typically one-, two-, or three-dimensional.
    \item A finite set of states $\mathcal{S}$ that each cell can occupy.
    \item A neighborhood function $\mathcal{N}$ that defines which cells influence a given cell.
    \item An update rule $f: \mathcal{S}^{|\mathcal{N}|} \rightarrow \mathcal{S}$ that computes the next state \cite{IlachinskiCellular}.
\end{itemize}
